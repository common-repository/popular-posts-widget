Tags: widget, popular posts, post, comments
Tested up to: 3.4.2
Requires at least: 2.8
Stable tag: 1.5

== Description ==

This plugin will provide you a new widget which shows the most popular posts in a unordered list.
Widget options:Title, Amount of posts, Custom ul class, Custom li class, post type
